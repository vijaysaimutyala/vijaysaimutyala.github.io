pipreqs

pipreqs solves the issues of what packages you need to install. Once in a while you download a random project from GitHub or put together a project quickly without caring for a requirements.txt file. Now when you want to publish this app online, this becomes a challenge. pipreqs saves you here.

pip install pipreqs
pipreqs your_project_root_folder

Now the second command, generates a requirements.txt file in you project root folder üòÑ

jupyter notebook config generation

The below command lets you generate the jupyter config file usually in C:\Users\vijsu\.jupyter\jupyter_notebook_config.py location on Windows. One of the main reasons I use this is when I'm setting up jupyter on a new machine, I always make sure that all my notebooks are in same place. 

jupyter notebook --generate-config

## The directory to use for notebooks and kernels.
c.NotebookApp.notebook_dir = 'E:\Jupyter Notebook'
